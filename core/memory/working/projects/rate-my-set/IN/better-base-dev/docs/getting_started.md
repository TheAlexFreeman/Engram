# Getting Started

There are a lot of interconnected pieces and parts (and ideas) in this codebase.

For starters, I'd recommend that you get the following things up and running, and confirm at the end that you're able to see things working and running.

From there, I'd recommend you play with the starter template for a bit and get a sense of what's there. If you'd like, peek under the hood and see how things are working. However, the guide will probably be the most helpful for getting new things up and running.
