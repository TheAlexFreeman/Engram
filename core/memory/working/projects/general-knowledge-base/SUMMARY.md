# Project: General Knowledge Base

## Description
Accumulate broadly useful knowledge on the user's behalf, especially in areas they revisit often or are actively learning. This is the open-ended research home for knowledge that is durable but not tightly bound to a finite delivery project.

## Cognitive mode
Exploration mode means the agent should map the user's domain landscape, notice repeated retrieval patterns, and propose the highest-leverage next knowledge investments.

## Artifact flow
- IN/: project-scoped research notes, candidate topic maps, and synthesis drafts that are not yet ready for global publication
- OUT contributions: vetted artifacts that are broadly reusable or ready for promotion into the main knowledge base

## Notes
This project is perpetual. It should remain lightweight and opportunistic rather than turning every curiosity into a formal research campaign.
